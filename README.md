# SwimTrack

Swimming performance tracker — log meet results, training sessions, goals, and compare against USA Swimming motivational standards.

**Live site:** [https://poohandmouse.github.io/swimtrack/](https://poohandmouse.github.io/swimtrack/)

## Local development

```bash
npm install
npm run dev
```

## Deploy

Pushes to `main` automatically deploy to GitHub Pages via `.github/workflows/deploy.yml`.
